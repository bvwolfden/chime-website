# Chime Website

Static website for Chime - AI Task Reminders app.

Hosted at: **https://prvail.ai**

## Pages

- **index.html** - Homepage
- **privacy.html** - Privacy Policy (required for App Store)
- **support.html** - Support & FAQ

## GitHub Pages Setup

This site is deployed via GitHub Pages:
- Custom domain: prvail.ai
- HTTPS enabled
- Deploys automatically on push to main

## Local Development

Just open the HTML files in a browser. No build process needed.
